## 便于安装

安装方法
1. 向cnblogs申请JS权限

2. 将`博客皮肤`改为`SimpleMemory`

3. 下载`style.css`、`page_footer.html`、`bar.html` / 下载release中的simple-theme.zip

4. 将`style.css`中的内容复制到`页面定制 CSS 代码`

5. 将`side_bar.html`中的内容复制到`博客侧边栏公告`

6. 将`page_footer.html`中的内容复制到`页脚 HTML 代码`

7. 保存
